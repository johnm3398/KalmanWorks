# KalmanWorks C++

This directory is reserved for the planned C++ implementation of KalmanWorks inside the shared
`implementations/` umbrella.

Status: in development.

The intent is to build a C++ codebase that is semantically aligned with the Python implementation
once the core model contracts, estimator abstractions, and logging concepts have stabilized well
enough to mirror cleanly.

This folder exists intentionally so the repository can separate language-specific project roots
from the shared notebooks and top-level documentation.
