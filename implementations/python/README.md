# KalmanWorks Python

This directory is the Python project root for KalmanWorks inside the shared
`implementations/` umbrella.

The package code lives under `src/kalmanworks/`, while `pyproject.toml` defines the Python build,
packaging, and dependency metadata for this implementation track.

As notebook-grown abstractions stabilize, they can be migrated here in a form that is reusable,
testable, and ready for packaging.
